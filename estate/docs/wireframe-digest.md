# Wireframe: Digest (Desktop) v0.1

> VERITY ESTATE **Digest** 페이지. 내부 리포트 + 공개 출력 제작·발행·감사.
> 데스크톱 전용. 내부 사용자 전용.
> 이 페이지는 `dilution-rules.md`의 **실행 UI** — 희석 필터와 발행 큐를 직접 조작.

---

## 1. 설계 결정

- **2개 탭으로 명확히 분리**: `[🔒 내부]` / `[🌐 공개]` — 두 맥락이 섞이면 실수로 내부 콘텐츠를 공개 경로로 보낼 위험.
- **발행 전 자동 체크리스트** — `dilution-rules.md § 5.2` 6개 항목을 UI로 강제.
- **변환 로그 뷰어** — 과거 발행 이력 전수 조회. 감사 대응.
- **인시던트 로그** 별도 서브탭 — L1~L3 유출 사건 추적.

**핵심 요소**:

- **좌**: 발행 큐 (상태별: 🟢 예약됨 / 🟡 초안 / ⚪ 보류 / 아카이브)
- **중**: 에디터 — 인스타 카드뉴스 / 뉴스레터 / 블로그 포맷 선택
- **우**: **Dilution Check** — `dilution-rules.md § 5.2` 6개 항목을 **자동** 검증

**에디터 채널별 기능**:

| 채널 | 에디터 모드 |
|---|---|
| 인스타 카드뉴스 | 카드 3~5장 템플릿. 드래그·리오더. 이미지 프레임·폰트 고정 |
| 뉴스레터 | 마크다운 에디터. 섹션 6종(매크로·주목지역·호재·리스크·데이터해설·공지) |
| 블로그 | 뉴스레터와 동일 콘텐츠 재활용 + 추가 섹션 |

**Dilution Check 상태**:
- 6개 항목 모두 통과 시 `[발행 예약]` / `[즉시 발행]` 활성화
- 하나라도 실패 시 해당 버튼 비활성화, **실패 사유 구체 표시**
- "지역 노출 이력" 섹션 — 해당 구가 최근 4주 내 어느 공개물에 등장했는지 표시

## 5. 자동화 스펙

- **내부 리포트 자동 생성**: 주간·월간·분기 cron. 최근 데이터·알림·VAMS 상태 취합
- **공개 초안 자동 생성**: 내부 리포트 → 희석 필터 → 카드뉴스 템플릿 (사용자 확인 후 발행)
- **Dilution Check 자동 실행**: 초안 저장 시마다 6개 항목 검증
- **발행 예약 실행**: 예약 시간에 자동 게시 (API — Instagram Graph, Mailchimp 등)
- **라벨 편향 알림**: 월 1회 통계 집계 후 경고
