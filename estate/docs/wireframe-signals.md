# Wireframe: Signals (Desktop) v0.1

> VERITY ESTATE **Signals** 페이지. 모든 알림의 전체 뷰 + 상세 검증 + 알림 룰 설정.
> 데스크톱 전용. 내부 사용자 전용.

---

## 1. 설계 결정

- **3칼럼**: 좌 카테고리 필터 / 중 알림 피드 / 우 선택 알림 상세.
- **"알림이 맞았나" 검증 탭** 내장 — 과거 알림의 적중 여부를 n일 경과 후 실제 추이와 비교. 모델 튜닝의 핵심 루프.
- **알림 설정(룰 편집)은 같은 페이지 내 Drawer**로 — 별도 페이지로 분리하면 룰 수정 후 결과 확인이 단절됨.

## 3. 영역별 상세

### 3.1 좌 필터

| 필터 | 동작 |
|---|---|
| **Category** | 🔴 GEI / 🟡 Catalyst / 🟣 Regulation / 🔵 Anomaly / 전체 |
| **Watchgroup** | WatchGroups별 필터. 본인 관심 지역만 보기 |
| **Severity** | 높음 / 중간 / 낮음 — 카테고리 내부 등급 |
| **Time** | 7일 / 30일 / 90일 / 사용자 지정 |
| **Status** | 미확인 / 확인됨 / 숨김 (아카이브) |

### 3.2 중 피드 (알림 카드)

**카드 구조**:
```
[timestamp]
[icon] [구명] [요약]
       [변화량·근거 한 줄]
       [액션: 상세 | 숨김]
```

**정렬**: 기본 시간 역순. 옵션으로 Severity 순.

**숨김 처리**: 확인·무시한 알림은 숨김. `[📥 아카이브]`에서 복원 가능.

**하단 `[⚙ 알림 룰 편집]`**: Drawer 우측에서 슬라이드. 3.4 참조.

### 3.3 우 선택 알림 상세

**구성 (위→아래)**:

**1) 트리거 조건 원문**
- 어떤 룰이 이 알림을 발생시켰는지 (예: "GEI 40 돌파 + 구 내 주요 상권 Rent ∆ > 10%")

**2) 근거 피처 기여도**
- 막대 그래프로 표시. GEI 변화 +4 중 Rent +12, Merchant +8, … (내부용 풀디테일)

**3) 검증 탭 (핵심)**
- 발생 후 n일 경과 데이터로 "이 알림이 맞았나" 검증
- 7일·14일·28일·90일 후 실제 추이와 비교
- 예: "28일 후 해당 상권 Rent +8.2% 추가 상승 → 적중"
- 과거 유사 알림 이력 함께 표시 (**n-shot 학습 자료**)

**4) 관련 페이지 이동**
- `[지역 탐색 → 해당 구]` — 지도로 점프
- `[VAMS 포지션 생성]` — 이 신호 기반 가상 매수
- `[공개 발행 초안]` — Digest 페이지에서 이 알림을 공개 콘텐츠로 변환 시작

### 3.4 알림 룰 편집 Drawer

`[⚙ 알림 룰 편집]` 클릭 시 우측 슬라이드.

**구성**:
- **전역 룰**: 임계치 조정 (예: GEI 40 돌파 → 38로 낮춤)
- **카테고리별 on/off**
- **WatchGroups별 별도 룰**: "출퇴근권에서만 중간 severity 이상 알림"
- **채널**: 데스크톱 배너 / 이메일 / 웹훅 / 텔레그램 봇
- **Quiet hours**: 심야 알림 억제

**프리셋**: "보수적 (민감도 낮음)" / "표준" / "적극적 (민감도 높음)" 세 세트 빠른 전환.

## 4. 마스킹·유출 방지

| 영역 | 기본 | Privacy Mode ON |
|---|---|---|
| 알림 카드 변화량 숫자 | 노출 | 방향성만 (+/−) |
| 근거 피처 기여도 그래프 | 노출 | 숨김 |
| 트리거 조건 원문 | 노출 | 카테고리명만 |
| 과거 유사 알림 수치 | 노출 | 적중 여부(✓/✗)만 |

**가장 위험한 유출 경로**: 근거 피처 기여도 막대 그래프 → 가중치 역산 가능. 스크린샷·캡처 극도 주의.
