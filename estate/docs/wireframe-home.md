# Wireframe: Home Dashboard (Desktop) v0.1

> VERITY ESTATE 홈 대시보드. **데스크톱 전용**.
> 모바일 레이아웃은 별도 문서 `wireframe-mobile.md`에서 다룬다.
> 이 화면은 **내부 사용자 전용**. 공개 채널로 스크린 공유 시 VAMS 영역 마스킹 필수.

---

## 1. 설계 결정

**3칼럼 절충안 채택**:
- **좌**: AlertDashboard (매일 여는 신호)
- **중**: Today's Focus + WatchGroups + VAMS (메인 어텐션)
- **우**: 미니맵 + Macro + 발행 예약

"매일 아침 열었을 때 뭐가 먼저 보여야 하는가" = **신호 + 관심그룹 상태**.
지도는 탐색용이라 축소 배치, 클릭 시 전체 지도 페이지로 확장.

기준 뷰포트: **1440px** (13" MBP 기본 외부 디스플레이 가정).
최소 지원: 1280px (좌 칼럼 축소).

## 3. 영역별 상세

### 3.1 Top Nav

```
VERITY ESTATE    [⌘K Search]        [🔔]  [⚙]  [User]
Home  지역 탐색  Signals  VAMS  Macro  Digest
```

- **⌘K Search**: 구·상권·호재명 빠른 검색 (Cmd+K)
- **🔔 아이콘**: 미확인 알림 배지. 클릭 → AlertDashboard 확장 뷰
- **탭**: 6개. `Home`은 현재 페이지 하이라이트

### 3.2 Left: AlertDashboard

**목적**: 매일 아침 "어제 이후 뭐가 바뀌었나"를 30초에 파악.

**알림 카테고리 (색상 구분)**:

| 색 | 카테고리 | 트리거 |
|---|---|---|
| 🔴 | GEI 경보 | Stage 전환 (1→2, 2→3), 특히 **40 돌파 알림** |
| 🟡 | 호재 업데이트 | 교통·재개발 단계 배수 변화 |
| 🟣 | 규제 변화 | 투기지역 지정·해제, 전매·대출·세제 관련 뉴스 |
| 🔵 | 이상거래 | 실거래 통계적 이상치 (동 단위 YoY +XX% 단일건) |

**항목 구조**:
```
[아이콘] [구/상권명] [이벤트 요약]
         [상세 한 줄]
```

**인터랙션**:
- 클릭 → 우측 Drawer 슬라이드. 상세 카드 (트리거·근거 피처·스코어 변화)
- 읽음 처리: 자동 (클릭 시) + 수동 토글
- 하단 "전체 알림 →" 클릭 → Signals 탭 전체 알림 로그로 이동

**밀도 기준**: 미확인 10건 이상이면 스크롤. 과거 알림은 3일까지 표시 후 아카이브.

### 3.3 Center Top: Today's Focus

**목적**: 오늘 가장 주목해야 할 단일 이벤트를 크게 띄움.

**선정 로직** (우선순위):
1. 새로 발생한 GEI Stage 2 전환 (조기경보) — 최우선
2. 관심그룹 내 LANDEX 등급 상향·하향
3. 관심그룹 내 Catalyst 단계 진입
4. 위 전부 없으면 "전체 시장 스냅샷" 모드로 대체

**카드 구조**:
```
⭐ 성동구 GEI Stage 2 전환 (성수동 상권 중심)
4주 전 대비 Rent +18%, Merchant Turnover 가속
권고: 신규 진입 보류
[상세 →]
```

**내부 표시 OK**: 수치·Stage·권고 문구 모두 내부 뷰라 풀 표시.
단, **스크린샷 유출 시 L3 치명 유출** — 마스킹 토글 필수(3.8 참조).

### 3.4 Center Mid: WatchGroups

**목적**: 본인이 구성한 관심 지역 그룹별 상태를 한눈에.

**그룹 카드 구조**:
```
▸ 출퇴근권 (5 구)
  HOT 0 / WARM 2 / NEUT 3 / COOL 0 / AVOID 0
```

**인터랙션**:
- `▸` 클릭 → 그룹 펼침 (5개 구 개별 등급 + 최근 변화)
- 그룹명 클릭 → 그룹 전용 페이지 (구별 상세 + 추이 그래프)
- **하단 `[+ 그룹 추가]`** → 새 그룹 생성 (모달)

**그룹 정의 (v0.1 기본 3개, 사용자 편집 가능)**:
- 출퇴근권: 본인 직장 접근성 기준
- 노후용: 의료·녹지 인프라 기준
- 재건축 베팅: 사업 단계 ≥ 조합 설립 인가 구역 포함

**내부 화면**: 등급 뱃지 옆에 원점수 소숫점까지 표시(내부 용도). 공개 금지.

### 3.5 Center Bottom: VAMS (가상 포지션)

**목적**: 실제 매수 전 가설 검증 포지션 모니터링.

**표시 항목**:
- 활성 포지션 개수
- 누적 수익률 (마스킹 토글 기본 ON 권장)
- 최근 진입 포지션 1개
- `[VAMS 상세 →]` 버튼

**마스킹 규칙** (3.8 참조):
- **기본 상태에서 수익률·구명은 블러**
- 클릭 토글로 한시 해제
- 스크린샷 유출 시 최대 위험 영역 — 기본 블러 강제

### 3.6 Right Top: Mini Map

**목적**: 서울 전역 스캔의 진입점. 중심은 아님.

**표시**:
- 서울 25구 폴리곤 + LANDEX 등급 컬러 오버레이
- 호버: 구명 + 등급 툴팁
- 클릭: 지역 탐색 페이지 전체 지도로 확장 (줌인·패닝 가능)

**크기**: ~360×280px. 라벨·범례는 생략 (미니 버전). 전체 범례는 지역 탐색 페이지에서.

### 3.7 Right Mid: Macro Snapshot

**목적**: 거시 지표 한 줄 요약. 판단 보조.

**고정 4줄**:
```
기준금리  3.50%  ↔
전세가율  65.2%  ↗
미분양    감소   ↘
심리지수  회복   ↗
```

- 화살표: 직전 대비 방향 (↗↘↔)
- 클릭 → Macro 탭으로 이동

### 3.8 Right Bottom: Publish Queue

**목적**: 공개 발행 규율 유지. **이 자리는 제품 정체성의 일부** — 매일 "이번 주 내보낼 것"이 보여야 규율이 생김.

**표시**:
- 이번 주 예약된 포스트 (인스타 카드뉴스, 뉴스레터)
- 다음 달 뉴스레터 초안 상태 (⚠ 초안 없으면 경고)
- `dilution-rules.md § 5 발행 주기 제한` 근거

**인터랙션**:
- 항목 클릭 → Digest 페이지 해당 초안 열기

## 4. 마스킹·유출 방지

이 홈 화면은 내부 뷰지만, **스크린 공유·캡처 시 L2~L3 유출 위험** 영역이 3곳.

| 영역 | 위험 등급 | 기본 상태 | 토글 |
|---|---|---|---|
| Today's Focus 원점수·Stage | L3 | **마스킹 해제** (내부 작업용) | 수동 토글로 가림 |
| WatchGroups 원점수 | L3 | 마스킹 해제 | 수동 토글 |
| VAMS 수익률·구명 | L3 | **마스킹 ON (기본값)** | 클릭 시 해제 |

**[👁 Privacy Mode]** 버튼을 Top Nav에 배치. 한 번 클릭으로 **L3 영역 전부 일괄 블러**. 화면 공유·발표 직전 활용.

## 5. 데이터 바인딩

각 영역이 어떤 내부 소스를 땡기는지:

| 영역 | 소스 |
|---|---|
| AlertDashboard | AlertDashboard 엔진 (`scoring-landex.md § 4 업데이트 주기` + `scoring-gei.md § 8`) |
| Today's Focus | 선정 로직 엔진 (실시간 계산) |
| WatchGroups | 사용자 정의 그룹 + 각 구 LANDEX 최신 스냅샷 |
| VAMS | VAMS 엔진 (별도 문서, v0.2 정의 예정) |
| Mini Map | LANDEX 월간 스냅샷 (구 단위) |
| Macro Snapshot | 한은·통계청·KB 주간 데이터 |
| Publish Queue | Digest 초안 저장소 + 발행 스케줄러 |
