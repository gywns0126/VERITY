# Wireframe: VAMS (Desktop) v0.1

> **V**irtual **A**sset **M**anagement **S**imulator — 부동산 버전.
> 실제 매수 전 **가설 검증용 가상 포지션** 관리.
> 데스크톱 전용. 내부 사용자 전용. **L3 위험도 최고.**

---

## 1. 설계 결정

- VERITY TERMINAL의 VAMS와 **동일 철학** — 실거래 아닌 가상. 본인 판단 훈련용.
- **기본 상태에서 수익률·구명 전면 마스킹 ON** — 유출 시 본인 투자 전략 노출 위험.
- **가설 백테스트 탭** 별도 — "만약 조건 X일 때 전부 매수했다면" 일괄 시뮬레이션.
- **진입 근거 메모 필수** — 근거 없이 추가 불가. 포지션이 쌓인 뒤 패턴 분석 가능.

## 3. 영역별 상세

### 3.1 상단 Summary 스트립

- 활성 포지션 수 / 누적 수익률 / 평균 보유 / 승률
- **기본 마스킹 ON** — `●●.●%` 처리. `[마스킹 해제]` 클릭 시 30초 한시 해제 후 자동 재마스킹.
- `[+ 포지션 추가]` — 모달 (3.4 참조)
- `[📈 가설 백테스트]` — 별도 뷰로 전환 (3.5 참조)

### 3.2 좌 포지션 리스트

**탭**: `[Active]` / `[Closed]`

**카드 구조**:
```
[신호등] 구명 · 단지명
  수익률 ± · 보유일수
```

신호등: 🟢 수익 / 🟡 중립 / 🔴 손실.
기본 마스킹 ON 시에도 신호등·보유일수는 표시(유출 위험 낮음). 구명은 마스킹.

### 3.3 중 선택 포지션 상세

**구성**:
- **헤더**: 구·단지·진입 시점·가정 매수가·현재 시세·수익률
- **진입 근거** (본인 메모 + 당시 LANDEX·GEI 스냅샷)
- **청산 조건** (자동 트리거 룰)
- 액션: `[메모 편집]` `[청산 예약]` `[청산 (가상)]` `[삭제]`

**진입 근거 필드**:
- 자동 채워짐: 당시 LANDEX 등급·GEI Stage·주요 호재
- 수동 메모: 2~3줄 자유 입력 (필수)

**청산 조건** (OR 조건 복수 허용):
- LANDEX 등급 이하 진입 (예: COOL)
- GEI Stage 이상 진입 (예: Stage 3)
- 수익률 절대 임계치 (±%)
- 보유 기간 초과
- 특정 호재 소멸

### 3.4 우 차트·검증

**1) 시세 추이 차트**
- 2년 기간 가격 라인
- 진입가·현재가·목표가(선택) 마커
- 관련 호재 이벤트 마커

**2) 진입 당시 vs 현재 스냅샷 비교**
- LANDEX, GEI, 주요 호재 단계를 좌우 대조
- 변화량 시각적으로 표시

**3) 가설 검증 (회고)**
- 진입 당시 **가정**했던 시나리오와 **실제 전개** 비교
- 예: "당시 GEI Stage 1 유지 가정 → 실제 유지 ✓"
- 이 필드가 누적되면 **본인 판단의 정확도 자체가 데이터**가 됨

### 3.5 신규 포지션 추가 (모달)

```
[+ 포지션 추가]
──────
구 선택:        [드롭다운]
단지 (선택):    [검색]
가정 매수가:    [₩ 입력]
진입 근거 메모: [텍스트 필수, 2~3줄]
청산 조건:      [조건 빌더]
──────
[취소] [추가]
```

**진입 근거 메모 필수**. 근거 없는 충동 매수 시뮬레이션 방지.

### 3.6 가설 백테스트 탭 (별도 뷰)

**목적**: "만약 조건 X를 만족하는 모든 구를 매수했다면 수익률이?" 일괄 시뮬레이션.

**구성**:
- 조건 빌더: `LANDEX ≥ WARM AND GEI Stage ≤ 1 AND Catalyst 재개발 포함`
- 기간: 2019~현재
- 진입 타이밍: 매월 첫날 / 조건 충족 시점
- 청산 규칙: 보유 12M / LANDEX COOL 진입 시

**출력**:
- 시뮬레이션 포지션 리스트 (가상)
- 기간별 수익률 분포
- 승률·손익비·최대 낙폭

**이 결과는 절대 공개 금지**. 내부 방법론 튜닝용.

## 4. 마스킹·유출 방지

**이 페이지는 전면 L3.** 기본 마스킹 최고 수준.

| 영역 | 기본 (보통 모드) | Privacy Mode ON |
|---|---|---|
| Summary 스트립 수익률 | 마스킹 ON | 마스킹 유지 |
| 포지션 리스트 구명·수익률 | 마스킹 ON | 마스킹 유지 |
| 선택 상세 수치 전체 | **해제 시 30초 한시** | 마스킹 유지 |
| 차트 y축 가격 | 표시 | 숨김 |
| 가설 백테스트 결과 | 표시 | 통계 요약만 |

**스크린샷·녹화 절대 금지** 영역. 비공개 링크 공유 시에도 이 탭만큼은 제외 권장.

| 포지션 리스트 | VAMS 포지션 DB (본인 전용) |
