# Wireframe: Macro (Desktop) v0.1

> VERITY ESTATE **Macro** 페이지. 매크로 지표 해설.
> 데스크톱 전용.
> **공개 희석 위험도 낮음** — 공공 데이터 해설 중심이라 상당 부분 외부 공개 가능.

---

## 1. 설계 결정

- **경량 페이지**. 다른 페이지 대비 밀도·기능 단순.
- **공개 친화적**: 매크로 해설은 카드뉴스·뉴스레터 재료로 자주 차용됨. 내부 풀디테일과 공개 버전의 간극이 작음.
- **다만 "매크로 ↔ 부동산 연관 해설"은 내부 관점** — 공개 시 희석 필요.

## 3. 영역별 상세

### 3.1 상단 Key Indicators (4카드)

| 카드 | 데이터 | 출처 |
|---|---|---|
| 기준금리 | 현재 + 최근 변동 추세 | 한국은행 |
| 전세가율 | 전국·서울 평균 | KB 시세 |
| 미분양 | 전국 호수 + 3개월 추세 | 국토부 |
| 심리지수 | 주택매매심리지수 | 한국부동산원 |

- 각 카드 클릭 → 메인 차트 영역에 해당 지표 표시
- 화살표: 방향(↗↘↔) + 변화량
- **이 숫자들은 모두 공공 데이터** — 공개 희석 대상 아님

### 3.2 메인 차트

- 선택된 지표의 시계열 라인
- 기간 토글: 1Y / 3Y / 5Y / 10Y
- 정책 변경 이벤트를 차트 위에 **수직 마커**로 표시 (정책 타임라인과 연동)
- 복수 시리즈 비교 가능 (예: 전국 vs 서울)

### 3.3 정책 타임라인

- 역순(최신 상단) 스크롤
- 부동산 관련 정책·규제·조치 이벤트
- 각 항목 클릭 → 해당 시점으로 메인 차트 마커 이동

**데이터 수집**:
- 국토부·기재부 보도자료 RSS
- 수동 큐레이션 (중요 이벤트는 본인이 추가)

### 3.4 Macro ↔ Real Estate 해설 (⚠ 내부)

**이 섹션만 내부 관점**. 매크로 지표가 LANDEX·GEI에 어떻게 영향을 주는지 본인 판단을 서술.

**구성**:
- **현재 매크로 환경 요약**: 4대 지표를 종합 해석
- **LANDEX 방향성 영향**: 매크로 → 종합 스코어 방향성 판단
- **본인 메모 추가**: 자유 텍스트. 주간 업데이트용

**공개 희석 처리**:
- 일반적 해설(금리→심리) 수준은 공개 가능
- "LANDEX 상방 압력", "Catalyst 가중 지역 우위" 같은 **내부 스코어 참조는 공개 금지**
- Digest 페이지에서 공개 버전으로 변환 시 `dilution-rules.md § 2.1`에 따라 자동 필터

| 영역 | 기본 | Privacy Mode ON |
|---|---|---|
| Key Indicators 4카드 | 노출 | 노출 (공공 데이터) |
| 메인 차트 | 노출 | 노출 |
| 정책 타임라인 | 노출 | 노출 |
| **Macro ↔ RE 해설 (내부)** | 노출 | **숨김** (또는 요약만) |

이 페이지는 다른 페이지 대비 스크린 공유 부담 낮음. 단 하단 해설 섹션만 주의.

## 6. 공개 활용 (특이사항)

Macro 페이지 콘텐츠는 **Digest 공개 채널의 주간 카드뉴스 재료로 자주 차용**된다.

- Key Indicators 4카드 → "이번 주 매크로 한눈에" 카드 1장
- 정책 타임라인 상단 1건 → "이번 주 정책 업데이트" 카드 1장
- Macro ↔ RE 해설은 **희석 후** "이번 주 전체 시장 방향성" 코멘트로 변환

단, `dilution-rules.md § 5 발행 주기 제한` 준수 (주 1회).
