# Wireframe: Region Explorer (Desktop) v0.1

> VERITY ESTATE **지역 탐색** 페이지. 서울 전역 히트맵 + 드릴다운.
> 데스크톱 전용. 내부 사용자 전용 뷰.

---

## 1. 설계 결정

- **지도 중심 레이아웃** — 홈 대시보드는 알림 중심이지만, 지역 탐색은 지도가 메인.
- **좌 필터 / 중 대형 지도 / 우 선택 구 상세 / 하단 25구 테이블** 4분할.
- **타임 슬라이더** 포함 — 과거 스냅샷(월간) 재생 가능. 트렌드 판별에 결정적.

## 3. 영역별 상세

### 3.1 좌 필터 패널

| 필터 | 기본값 | 동작 |
|---|---|---|
| **Overlay** (radio) | LANDEX | 지도 색 체계 전환 |
| **Grade** (checkbox) | HOT·WARM·NEUT 선택 | 해당 등급 외 회색 처리 |
| **Subscore range** (slider) | 0~100 | 각 서브스코어 범위 필터 |
| **GEI Stage** (checkbox) | 2·3·4 | 리스크 단계별 필터 |
| **Catalyst** (checkbox) | 교통·재개발 | 호재 유형별 마커 표시 |
| **Reset** | — | 전체 초기화 |

- 필터 변경 시 지도·테이블 실시간 재채색·재정렬
- URL 쿼리스트링과 동기화 → 뷰 공유 가능 (내부 공유 링크에만)

### 3.2 중앙 지도

**Overlay 모드**:

| 모드 | 컬러 스케일 | 의미 |
|---|---|---|
| LANDEX | HOT(레드) → AVOID(회색) | 종합 등급 (기본) |
| GEI | Stage 0(투명) → Stage 4(보라) | 리스크 단계 |
| V·D·S·C·R | 각각 단색 그라데이션 | 서브스코어 개별 보기 |
| Catalyst | 구역 + 호재 마커 | 호재 밀도 시각화 |

**지도 인터랙션**:
- 호버: 구명 + 선택 overlay 값 툴팁
- 클릭: 해당 구 선택 → 우 패널 활성 + 지도 가볍게 줌인
- 더블클릭: 구 내 동 단위 확대 (v0.2 대비 placeholder)

**호재 마커**:
- 🚇 교통 (역 신설 위치에 아이콘)
- 🏗 재개발 (구역 폴리곤 외곽선)
- 🏛 정책 특구 (해당 구역 하이라이트)

### 3.3 Time Slider

```
◀ [2024.01 ●━━━━━━━━━━━━○ 2026.04] ▶
```

- 월간 스냅샷 재생 (2년치)
- `●` 위치로 드래그 → 해당 시점 스코어로 지도·테이블 재채색
- 재생 버튼: 초당 3개월 속도로 자동 재생 → **패턴 애니메이션 관찰**
- 특정 이벤트 마커 표시: "GTX-C 실시설계 진입", "규제 해제" 등

### 3.4 우 선택 구 상세

구 클릭 시 활성. 내부 풀디테일 표시.

**구성**:
- 상단: 구명 + LANDEX 등급·원점수 + GEI Stage·원점수
- **서브스코어 레이더 차트** (V·D·S·C·R 5축)
- 주요 호재 리스트 (단계 배수 포함)
- 최근 변화 타임라인 (30일 단위 이벤트)
- `[상세 리포트 →]` → 구별 심층 리포트 페이지 (v0.2)

**마스킹 토글**: 원점수·기여도는 L3. Privacy Mode ON 시 등급·Stage 라벨만 표시.

### 3.5 하단 25구 테이블

**컬럼**: 순위 / 구명 / LANDEX / 등급 / GEI / Stage / V / D / S / C / R / 30일 변화

- 기본 정렬: LANDEX 내림차순
- 컬럼 헤더 클릭 → 해당 기준 정렬
- row 클릭 → 지도 해당 구로 패닝·포커스 + 우 패널 활성
- `[CSV 내보내기]`: 내부 전용. 파일명에 `INTERNAL_ONLY` 접미 자동 부착
- `[스냅샷 저장]`: 현재 필터·타임·정렬 상태 북마크 (다른 사람과 비교용)

## 4. 마스킹·유출 방지

이 페이지는 **L3 위험도 최고** — 원점수·서브스코어·호재 단계 배수 전부 노출.

| 영역 | 기본 상태 | Privacy Mode ON |
|---|---|---|
| 지도 overlay 색상 | 노출 | 등급만 (숫자 없음) |
| 우 패널 원점수·레이더 | 노출 | 등급·Stage 라벨만 |
| 테이블 수치 컬럼 | 노출 | `—` 표시로 가림 |
| CSV 내보내기 | 가능 | **비활성화** |

**화면 공유 시**: Privacy Mode 강제 ON. 이 페이지는 외부 발표·녹화에 가장 부적합한 뷰.
