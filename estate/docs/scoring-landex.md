# LANDEX Score v0.1

> VERITY ESTATE 내부 스코어. 서울 25개구 단위 지역 종합 평가.
> 이 문서는 **내부 전용**. 공개 출력은 `dilution-rules.md` 변환 규칙을 거친다.

---

## 1. 목적

서울 25개구에 대해 **"지금 이 지역에 진입할 가치가 있는가"**를 0~100 단일 수치로 요약한다. 단지별이 아니라 **지역(구) 단위** 스코어. 단지 검색은 직방·호갱노노에 위임하고, VERITY ESTATE는 "어느 구에 들어갈지"의 층위에서 의사결정을 지원한다.

포지셔닝: **현재 시세 리포터 아님. 미래 가치 조기 포착기.**

---

## 2. 산출식

```
LANDEX = 0.30·V + 0.25·D + 0.15·S + 0.25·C − 0.05·R

V = Value Dynamics        (가치 동학)
D = Demand Pressure       (수요 압력)
S = Supply Dynamics       (공급 역학)
C = Catalyst Density      (호재 밀도)
R = Risk Discount         (리스크 감점)
```

- 각 서브스코어는 **0~100**으로 정규화
- 정규화 방식: **서울 25구 내 percentile rank → 0~100** (절대값 아닌 상대순위)
- Risk는 **감점** 형태로 상단에서 차감 (별도 카테고리가 아니라 penalty)
- 최종 LANDEX는 0~100 범위. 이론상 음수 가능하나 Risk 감점 5%는 작아서 실무상 0~100.

## 3. 서브스코어 구성

### 3.1 Value Dynamics (30%)

| 피처 | 가중치 | 출처 |
|---|---|---|
| 실거래가 12M 모멘텀 | 35% | 국토부 실거래가 API |
| 공시지가 vs 실거래 괴리율 | 25% | 국토부 공시지가 + 실거래 |
| 전세가율 (구간함수) | 25% | KB 시세 / 실거래 |
| 거래량 변화율 (YoY) | 15% | 실거래 건수 집계 |

**전세가율 구간함수 설계**:
너무 낮으면 매매 버블 신호, 너무 높으면 갭투자 과열 신호 — 단조함수로 쓰면 안 됨.

```
전세가율 < 50%  → 20점   (매매가 과열 위험)
50~60%          → 50점
60~70%          → 80점   (건강한 구간)
70~80%          → 100점  (안정적 실수요)
80~90%          → 70점   (갭투자 취약)
> 90%           → 30점   (역전세 위험)
```

### 3.2 Demand Pressure (25%)

| 피처 | 가중치 | 출처 |
|---|---|---|
| **청약 경쟁률** (1순위 지표) | 40% | 청약홈 공공 API |
| 인구 순유입 (YoY) | 25% | 통계청 주민등록 인구 |
| 중위소득 변화 | 15% | 통계청 가계동향 |
| 대중교통 접근성 (지하철 역 밀도·환승 노드) | 20% | 서울시 교통 API |

**청약 경쟁률 처리 주의**:
- 분양가상한제 적용 단지는 수요와 무관하게 경쟁률 폭발 → **분양가/주변 실거래 시세 배율**로 정규화
- 로또분양 필터: 배율 0.7 이하 단지는 가중치 ×0.5로 디스카운트
- **구축 밀집 구 fallback**: 해당 연도 청약 건수가 임계치 미만인 구는 **갱신 계약 전환율**(전세→매매 전환 비율)로 대체

**제거한 피처**: 네이버 부동산 검색 트렌드 — 후행 신호이고 수집 안정성 낮음.

### 3.3 Supply Dynamics (15%)

| 피처 | 가중치 | 출처 |
|---|---|---|
| 분양·입주 물량 (향후 2년) | 40% | HUG / 국토부 |
| 미분양 수준 | 35% | 국토부 미분양 통계 |
| 건축허가 증감 (YoY) | 25% | 국가통계포털 |

**방향성 주의**: Supply는 역방향 지표. 공급 많을수록 스코어 낮아짐.

### 3.4 Catalyst Density (25%) ⭐ 핵심 차별화

```
C = Σ (호재별 기본점수 × 단계 배수)
```

**교통 호재 단계 배수**:
| 단계 | 배수 |
|---|---|
| 검토·기획 발표 | ×0.3 |
| 예비타당성 통과 | ×0.6 |
| 실시설계·착공 | ×0.85 |
| 공사 진행 중 | ×1.0 |
| 개통 완료 | ×0.5 (이미 반영됨) |

**재개발·재건축 단계 배수**:
| 단계 | 배수 |
|---|---|
| 정비구역 지정 | ×0.2 |
| 조합 설립 인가 | ×0.4 |
| 사업시행 인가 | ×0.65 |
| 관리처분 인가 | ×0.85 |
| 이주·철거 | ×1.0 |
| 착공·분양 | ×0.8 |
| 입주 완료 | ×0.3 |

**호재 기본점수 (예시)**:
- GTX 역 신설: 30
- 지하철 신설(경전철·2호선 분기 등): 15
- 재개발 구역(구 내 1개 구역 기준): 10
- 대규모 재건축(단지 기준): 8
- 정부 규제 해제·특구 지정: 12

구 내 호재 점수 합산 → 25구 percentile → 0~100 정규화.

**v0.1 단순화**: 호재가 "해당 구 내 위치" 이진값으로 처리. v0.2에서 거리 감쇠(distance decay) 도입.

### 3.5 Risk Discount (5% 감점)

감점 요인:
- **GEI 스코어 임계치 근접도** (젠트리피케이션 조기경보 — `scoring-gei.md` 참조)
- 투기지역·조정대상지역 지정
- 전매 제한·대출 규제 강화
- 임대차법·세제 변동 영향

```
R = max(0, GEI_risk + 규제_risk + 세제_risk)
각 요소 0~100 → 가중합 후 5% 감점 스케일로 변환
```

## 4. 업데이트 주기

| 서브스코어 | 주기 | 트리거 |
|---|---|---|
| Value | 월간 | 실거래 데이터 갱신 |
| Demand | 월간 | 청약 결과 발표 |
| Supply | 월간 | HUG 월간 통계 |
| **Catalyst** | **이벤트 트리거** | 호재 발표·단계 변화 즉시 |
| Risk | 주간 | GEI 재계산 + 규제 모니터링 |

## 5. 공개 출력 (5등급 변환)

**내부 원점수는 절대 공개하지 않는다.** 등급만 외부 노출.

| 내부 점수 | 공개 라벨 | 공개 코멘트 톤 |
|---|---|---|
| 80+ | HOT ZONE | "강한 진입 신호" |
| 65~79 | WARM | "주목할 흐름" |
| 45~64 | NEUTRAL | "중립" |
| 30~44 | COOL | "관망" |
| <30 | AVOID | "주의 구간" |

**공개 금지 정보** (상세는 `dilution-rules.md`에서 확정):
- 원점수 (87.3 같은 수치)
- 서브스코어 분해
- 가중치 (0.30·V 같은 수식)
- 피처별 기여도
- 단계 배수 테이블
- 호재 기본점수

## 6. 검증 계획 (v0.2 → v1.0)

**백테스트**:
- 대상 기간: 2019~2024 (5년)
- 방법: 매년 1월 시점 LANDEX 재계산 → 해당 연도 말 수익률(매매가 상승률 + 전세 수익률) 순위와 LANDEX 순위의 상관계수 측정
- 목표: Spearman ρ ≥ 0.55 (강한 양의 상관)
- 실패 시 가중치·피처 재조정

**팩터 기여도 분석**:
- 각 서브스코어가 최종 성과 예측에 얼마나 기여했는지 회귀 분석
- 기여도가 0에 가까운 피처는 제거·대체
