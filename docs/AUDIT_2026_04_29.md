## 비판 1 — Brain Score 보너스 가산식 / 시그널 상관관계 무시

### 현행 공식 (`api/intelligence/verity_brain.py` line 2373)

```python
raw = (fs * w_fact + ss * w_sent
       + vci_bonus + gs_bonus + candle_bonus + inst_bonus
       - red_flag_penalty)
brain_score = round(_clip(raw, 0, 100))   # raw_brain_score 별도 보존
```

### 보너스 4개 정량 범위 (직교화 작업 사이즈 산정용)

| 보너스 | 함수 | 범위 | 트리거 조건 |
|---|---|---|---|
| `vci_bonus` | inline (line 2337~2341) | **-10 ~ +5** | VCI > 25 + fs ≥ 60 → +5 / VCI < -25 + fs < 50 → -10 |
| `gs_bonus` | `_compute_group_structure_bonus()` (line 2176) | **-3 ~ +5** | major_shareholders 집중도 / NAV 할인 4가지 룰 가산, clip(-3, 5) |
| `candle_bonus` | `_compute_candle_psychology_score()` (line 392) | **약 -3 ~ +3** | volume_bonus ±1.5 + rsi_bonus + 기타 candle 패턴 |
| `inst_bonus` | inline (line 2353~2370) | **0, 1, 3** | US 13F iscore ≥ 70 → +3 / ≥ 60 → +1 / 한국 종목엔 0 |
| `red_flag_penalty` | inline (line 2372) | **0 ~ 20** | downgrade_count × 5, max 20 |

**총 보너스 합 이론 범위**: -16 ~ +16. fs×w_fact + ss×w_sent (0~100) + 보너스 → raw 이론 -16 ~ 116, clip(0, 100).

### `w_fact` / `w_sent` 동적성 (이미 정적이 아님)

`decision_tree.brain_weights` 의 quadrant 별 분기:

| 분면 (성장 × 인플레이션) | fact | sentiment |
|---|---|---|
| 성장↑ 인플↓ | 0.65 | 0.35 |
| 성장↑ 인플↑ | 0.70 | 0.30 |
| 성장↓ 인플↓ | 0.80 | 0.20 |
| 성장↓ 인플↑ (수축) | 0.85 | 0.15 |
| default | 0.70 | 0.30 |

**비판 2 일부 부정확**: "sentiment 30%" 는 `default + 성장↑ 인플↑` 분면에서만. 수축기 자동 0.15 까지 감소. 정적 30% 가 아닌 동적.

### 직교화 작업 사이즈 추정

- 보너스 4개의 공분산 매트릭스 (작은 크기, 4×4)
- 백테스트 데이터에서 `vci/gs/candle/inst` 시그널 시계열 추출 → corr 매트릭스
- 가산 → orthogonal projection 변환 (Gram-Schmidt 또는 PCA)
- 영향 범위: `_compute_brain_score()` 내부만, 호출자 변경 없음
- **추정: 1~2일 (코드) + 1주 (백테스트 검증)**
